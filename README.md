Spring mvc maven project
Environment

slf4j 1.7.25 
Guide

Creating new oracle local db localhost:xe:1521 with username:c##student, password:student

Run dbscript/gameOjirappersDB.sql on c##student

Open or Import the project and reload pom.xml to reload maven files

Get Tomcat and navigate to "Build, Execution, Deployment", "Application Servers" and add "Tomcat Server".

In the "Deployment" tab, click the "+" button and select the Artifact.

Run Tomcat server.
